## PRODUCTION NOTES

### Recommended Tools for AI Video Generation:
1. **Synthesia** — Best for professional presenter videos with AI avatars
2. **HeyGen** — Great for branded avatar videos with custom voiceovers
3. **Runway Gen-3** — Text-to-video, good for screen recording simulations
4. **D-ID** — Animated talking head videos

### Voiceover Options:
- Use platform's built-in TTS (Synthesia, HeyGen)
- Generate with: ElevenLabs, Google Cloud TTS, or similar
- Keep voice professional but approachable (British or neutral American accent)

### Screen Mockups Needed:
- Landing page design (hero + value props)
- Browse/search interface
- 4-step contribute form (all screens)
- Success confirmation
- Curator dashboard
- Draft documentation editor
- Publish success state
- Public documentation page

### Color Palette Suggestions:
- Primary: Clean blue or teal (#0066cc or similar)
- Accent: Green for success states
- Neutral grays for UI backgrounds
- White/light backgrounds for readability

### Tone & Pacing:
- Upbeat, professional narration
- Smooth transitions between scenes (1-2 second fades)
- Cursor movements should feel natural (not too fast)
- Allow ~1-2 seconds for user to "read" each screen before action
- Subtle background music (optional) — focus on voiceover clarity
